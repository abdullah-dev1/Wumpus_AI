# Wumpus_AI

AI-based Wumpus World simulation using Knowledge-Based Agent, Propositional Logic, and Resolution Refutation with interactive frontend visualization.

AI-based Wumpus World simulation implementing a Knowledge-Based Agent using:

- Propositional Logic
- CNF Knowledge Base
- Resolution Refutation
- Interactive Web Frontend

## Features
- Interactive Wumpus grid visualization
- Logic-based safe cell inference
- Breeze and Stench percept detection
- Gold search simulation
- Real-time metrics dashboard
